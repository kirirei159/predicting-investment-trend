# predicting investment trend

## introduction

透過機器學習的方法去探討影響客戶持有各類金融商品的資產總額的因素，利用分析結果來訂定未來推廣金融商品所需要的策略。
期望透過洞察數據來發現潛在資訊並開創新的商業模式，期望能促使客戶獲得有效且即時的資源及服務，藉此提升客戶端的滿意度以及吸引新的客源。

## approach
1. 進行遺失值處理
2. 類別變數處理 - 讀熱編碼
3. 異常值處理
4. 新增特徵變數
5. model building
